# NOWHAE — 성장 피드

> **"어제보다 0.1% 나아진 오늘을 기록하세요"**  
> nowhae.co.kr 도메인 기반, 누구나 자유롭게 성장 기록을 올리고 응원할 수 있는 커뮤니티 피드

---

## ✅ 구현 완료 기능

| 기능 | 설명 |
|------|------|
| 피드 작성 | 닉네임·카테고리·연속일수·내용·이미지URL 입력 후 업로드 |
| 피드 조회 | 카드 그리드 형태로 전체 피드 표시 (최대 200개 로드) |
| 카테고리 필터 | 전체 / 공부 / 운동 / 독서 / 업무 / 습관 / 기술 / 기타 |
| 정렬 | 최신순 / 인기순(좋아요) / 연속일수 순 |
| 검색 | 닉네임·내용 키워드 실시간 검색 |
| 좋아요 | 하트 버튼으로 응원 / 로컬스토리지로 중복 방지 |
| 페이지네이션 | 9개씩 "더 보기" 방식 |
| 통계 배너 | 총 기록 수 / 총 응원 수 / 최장 연속일 |
| 스켈레톤 로딩 | 데이터 로드 전 shimmer 애니메이션 |
| 반응형 | 모바일 / 태블릿 / 데스크톱 완전 대응 |
| 토스트 알림 | 작성 완료·오류 등 피드백 알림 |

---

## 📂 파일 구조

```
index.html          ← 메인 페이지
css/
  style.css         ← 전체 스타일
js/
  main.js           ← 피드 CRUD + 좋아요 + 필터/정렬/검색 로직
```

---

## 🔗 API 엔드포인트

| Method | Path | 설명 |
|--------|------|------|
| GET    | `tables/posts?limit=200` | 전체 피드 조회 |
| POST   | `tables/posts` | 새 피드 작성 |
| PATCH  | `tables/posts/{id}` | 좋아요 수 업데이트 |

---

## 🗄️ 데이터 모델 (posts 테이블)

| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | UUID (자동) |
| nickname | text | 작성자 닉네임 |
| content | rich_text | 성장 내용 |
| category | text | 공부/운동/독서/업무/습관/기술/기타 |
| day_count | number | 연속 도전 일수 |
| likes | number | 좋아요 수 |
| image_url | text | 첨부 이미지 URL (선택) |
| created_at | datetime | 자동 생성 |
